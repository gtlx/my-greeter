# 安装与启动

## 1. 给 greeter 用户进入家目录的权限

```bash
sudo setfacl -m u:greeter:x /home/gtlx
```

这样 `greeter` 用户只获得穿过 `/home/gtlx/` 的权限，不能读、不能写。脚本放在用户目录内也能正常运行。

## 2. 编辑 greetd 配置

```bash
sudo vim /etc/greetd/config.toml
```

改为：

```toml
[terminal]
vt = 1

[default_session]
command = "/home/gtlx/软件/my-greeter/my-greeter.py"
user = "greeter"
```

## 3. 停用 lemurs，启用 greetd

```bash
sudo systemctl disable --now lemurs
sudo systemctl enable --now greetd
```

如果之前启动失败过，先重置：

```bash
sudo systemctl reset-failed greetd.service
sudo systemctl start greetd.service
```

## 4. 查看日志

```bash
journalctl -u greetd.service --no-pager | tail -20
```

## 5. 测试

按 `Ctrl+Alt+F1` 切到 TTY1，应该能看到登录界面。

## 回滚（恢复 lemurs）

```bash
sudo systemctl disable --now greetd
sudo systemctl enable --now lemurs
```
